This program will sum/subtract the sh** out of your text files!

///To compile yourself:

1. With Visual Studio Open AstroGrator.sln
2. Build with Release configuration
3. You will have working exe file

///Usage:

1. run cmd.exe
2. enter command: astrogrator.exe file1.txt +[-] file2.txt 
OR you can simple run astrogrator.exe and it will guide you through the process!
3. result will be calculated in file Result.txt, in the same dir 
as the exe is located
